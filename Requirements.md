# Application Overview
This application is to give the ability to track job applications and give people the opportunity to look at CV and cover letter templates. It should be clean and modern.

# User Flow
1. A user will have to sign up and log in with a local user name at first.
2. They should then be presented with a bold home page with 3 large icons - Job Tracker, Templates, Useful Advice
3. Each page should be able to navigate to one another

# Pages
## Job Tracker
- Kanban board style job tracker
- Columns to include Saved, Started, Applied, Interview, Successful, Unsuccessful
- Items should be able to be created with a Job title, link to job, application date, key contacts, salary expectations

## Templates
- This should be big and bold with templates to download

## Useful advice
- Should be a simple clean page with helpful job tips

# Build
- To be built on an Azure App service
- Use github actions to build the pipeline usingthe .github folder and modify where neccessary#
- It should use Node JS 24
